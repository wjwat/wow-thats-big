Wow, That's Big!

An API to return a single item for you to determine the size of it in bananas!

We've chosen two of the worst units of measurements for our items. Have fun figuring it out, suckers!

  • Weight is in bags of cement (50 kilograms)
  • Length is in Valve Hammers (16 Hammers = 1 foot)
